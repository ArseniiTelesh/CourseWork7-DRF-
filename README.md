## CourseWork7 (DRF)

Контекст В 2018 году Джеймс Клир написал книгу «Атомные привычки», которая посвящена приобретению новых полезных привычек и искоренению старых плохих привычек. Заказчик прочитал книгу, впечатлился и обратился ко мне с запросом реализовать трекер полезных привычек.

В рамках учебного курсового проекта реализовал бэкенд-часть SPA веб-приложения.

## Сборка образов

docker-compose build

## Запуск контейнеров

docker-compose up

## Запуск контейнеров в фоне

docker-compose up -d

## Сборка образа и запуск в фоне после успешной сборки

docker-compose up -d —build

## Выполнение команды внутри контейнера

docker-compose exec
